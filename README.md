### [SHOW-Q描述](https://github.com/OS-Q/SHOW-Q/wiki) 

[SHOW-Q](https://github.com/OS-Q/SHOW-Q)作为数据的最后展示，用于向用户提供最直观的表达。

[OS-Q](https://github.com/OS-Q)包括多种显示交互单元，这些呈现形式各有不同的组件，通过合理的搭配，完成系统的交互传达。
